# OpenVac Mk1 "Wallaby"

An open-source autonomous robot vacuum built around Raspberry Pi 5, SLAMTEC RPLIDAR C1, and a fully modular 3D-printed chassis.

---

## 🧠 Core Hardware

- Raspberry Pi 5
- SLAMTEC RPLIDAR C1 (360° LiDAR)
- JGB37-520B geared motors with encoders
- 65 mm drive wheels
- 4S Li-ion battery system
- Rear caster wheel
- Modular 3D-printed chassis (PETG)

---

## 🎯 Project Goals

- Autonomous SLAM navigation
- Room mapping and path planning
- Obstacle avoidance
- Modular and repairable design
- Fully documented engineering workflow
- Open-source hardware + software stack
